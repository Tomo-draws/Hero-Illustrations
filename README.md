# Hero Illustrations — Approved Asset Library

This repository contains the approved illustration library for Alliants design work.  
Maintained by **Tomo-draws**. All images are production-ready PNGs.

---

## For Claude Design

When using illustrations in mockups, layouts, or design concepts, **always source images from this repository**.  
Base URL for all assets:

```
https://raw.githubusercontent.com/Tomo-draws/Hero-Illustrations/main/
```

To use any image, append the filename:

```
https://raw.githubusercontent.com/Tomo-draws/Hero-Illustrations/main/Illo_Travel_Man_Senior_SuitAndTie_GreyHair_Pushing_Solo_StudioBackdrop_IndoorNoSeason_Commercial_Muted_LuggageCartDuffelBag_Professional.png
```

## Naming Convention

Files follow this structure:

```
Illo_[Category]_[Subject]_[AgeGroup]_[Style]_[Action]_[Group]_[Setting]_[Season]_[UseCase]_[ColourTone]_[Props]_[Mood].png
```

---

## How to Add New Illustrations

1. Export PNG from Illustrator / Photoshop
2. Name using the convention above
3. Upload via GitHub web interface (max 10 files per batch)
4. Add a new row to the index table above
5. Commit changes
